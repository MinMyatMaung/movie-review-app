# Movie Reviews Web App (TMDB + MongoDB)

A movie website that uses movie data from TMDB and lets users create/read/update/delete reviews stored in MongoDB.

## Project Structure

- movie-review-app
  - frontend: HTML, CSS, JavaScript (UI)
  - backend: Node.js, Express API, MongoDB

## Features
- Browse popular movies (TMDB)
- Search movies (TMDB)
- View reviews for a movie
- Create / update / delete reviews (MongoDB)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- External API: TMDB

---

## Setup

### Clone the repo
```
git clone <your-repo-url>
cd movie-app
```

### Install backend dependencies
```
cd backend
npm install
npm start
```

### Run the frontend
```Open `frontend/index.html` in your browser or use VS Code Live Server.```
